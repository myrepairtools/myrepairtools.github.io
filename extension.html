First-Time Install (Per Machine)

  1. Download the zip above and extract it to a permanent folder — e.g. Documents\myRepairTools. Don't run it from the Downloads folder.
  2. Open chrome://extensions and switch on Developer mode (top-right).
  3. Click Load unpacked and pick the folder you extracted.
  4. Pin myRepairTools to the toolbar (puzzle-piece icon → 📌).
  5. On the extension's Details page, turn on Allow access to file URLs — the Label Resizer needs it to read label PDFs you've downloaded and opened from the Downloads folder.

Updating a Machine

  1. Download the zip above again.
  2. Extract it over the same folder you installed from (replace all files).
  3. Open chrome://extensions and hit the ↻ reload arrow on the myRepairTools card.
Keep using the same folder path — Chrome ties the extension (and its saved settings) to that folder. Replacing the files and reloading updates it in place; deleting the folder and re-adding would reset its options.
