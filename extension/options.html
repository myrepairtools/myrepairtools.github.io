Latest update
Version 2.0.0

New features:

  • Rebranded as myRepairTools — one extension for the price calculator, RQ mods, and the new LCD buyback tools.
  • LCD Buyback: adding an iPhone / Galaxy S / Galaxy Note / Pixel screen repair to a ticket pops a Good/Bad display modal; every answer is logged to the MRT LCD Buyback Log.
  • Printing the ticket label also prints a send-display label (Dymo 30334) with the store, grade, model, ticket number, date and a scannable QR code for recycler audits.
  • Device families that trigger the modal can be toggled here — no update needed when new models come out.
