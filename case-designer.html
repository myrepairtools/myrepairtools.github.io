🔧 How this works (notes for the build)
  1. It's just two images stacked. Bottom layer: the customer's artwork, clipped to the case shape (a canvas here). Top layer: the phone mockup with a transparent window where the print area is, pointer-events:none so drags pass through. That's the whole Canva trick — same registration, same pixel size, art under, window on top.
  2. The top image must be a PNG with real alpha. The #1 way this fails: exporting the mockup layer as a JPEG (no transparency) or with a white background — then it just covers the art. From Canva/Photoshop: hide the mask/art layer, export PNG, keep the print window fully transparent.
  3. Both exports must be the exact same canvas size. One document, two exports (art-area guide + mockup overlay). If the sizes or positions differ even slightly, nothing lines up. Never crop one of them.
  4. Don't use the PSD in the browser. Browsers can't render PSDs — export flat PNGs from it. The PSD is only the master file.
  5. The iPhone 16 Pro model on this page IS the real manufacturer pair — the showroom photo (opaque JPEG) + the print mask (PNG whose black shape is the print area, everything else transparent). Both exported from the same 1500×1500 doc, so they register perfectly. The page draws the customer's art, clips it with the mask (destination-in), lays it over the photo, then multiplies the photo's own shading back over the print area so the art looks like it's under the clear case. Adding another real model is just two exports + one config entry:
assets: {
  photo: 'assets/cases/ip16pro-clear-photo.jpg', // showroom photo
  mask:  'assets/cases/ip16pro-clear-mask.png',  // print shape, alpha PNG
  px:    { w: 1500, h: 1500 },                   // the doc's pixel size
  print: { x: 448, y: 108, w: 604, h: 1268 },    // mask shape bbox, doc px
  cutoutPx: { x: 460, y: 112, w: 582, h: 352, r: 80 } // pop-out plate window
}
  1. Print files are physical, not screen-sized. Export renders at 300 DPI from the case's real millimeter dimensions, adds 2 mm bleed, and punches the camera opening transparent (the ring pops out for flat UV printing and reinstalls after — the opening itself is a hole).
